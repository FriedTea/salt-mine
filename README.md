# Salt Mine
![Screenshot](./screenshot.png)
Automatically download my salty dotfiles


## Build Dependencies
* ncurses
* curl
* libgit2

## Runtime Dependencies
* tar (For extracting tar.gz files)
* patch (For patching suckless builds)
